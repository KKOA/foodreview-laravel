## Setup Application
This insallation assumes have 
- MySQL
- PHP7
- Composer
- npm

installed.

```
git clone https://github.com/KKOA/foodreview-laravel.git
cd foodreview-laravel
composer update
php artisan key:generate
npm install
```

## Run Application
```
cd foodreview-laravel
php artisan serve
```
Click the link to open application in the browser
